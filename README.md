# Boston Housing Data Analysis

## 📌 About the Project
This project focuses on analyzing and preprocessing the Boston Housing dataset using Python.

## 🎯 Objective
- Analyze housing data
- Identify factors affecting house prices
- Prepare data for machine learning

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 🔍 Data Preprocessing
- Data cleaning
- Handling missing values
- Encoding categorical variables
- Feature engineering

## 📊 Analysis
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Data visualization

## 📈 Results & Insights
- Identified important features influencing house prices
- Discovered relationships between variables
- Prepared clean dataset for modeling
